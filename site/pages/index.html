GitHub Pages for convenience

The site uses GitHub pages as a convenient publishing and revision control platform. There's a couple of key resources below.

Getting started with pages

This is the pages site for the VeraPDF GitHub Organisation and isn't associated with a particular software project. The content for the site is generated from the master branch of a specially named GitHub repo https://github.com/verapdf/verapdf.github.io. The only purpose of this repo is to produce this site. GitHub pages also supports custom sub-domains so the URL can be whatever we wish.

For the GitHub pages basics read these posts, none of them are long. Anyone with permission can check out the repo make changes and check them back in, it only takes a few seconds to publish.

Although the site is static it's generated using Jekyll a ruby utility that processes text content with some templating magic. With loops, variables and basic data support you can get quite a long way with Jekyll.
